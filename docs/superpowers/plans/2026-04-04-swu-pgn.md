# SWU Portable Game Notation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a downloadable game log and SWU-PGN (Portable Game Notation) export to the Karabast game client, allowing players to download both a raw game log and a dual-layer `.swupgn` file after a game concludes.

**Architecture:** Server-side `SwuPgn` generator class transforms `GameChat.messages` and `Player.decklist` into the SWU-PGN format. A new `getGameLog` socket event returns both raw log text and `.swupgn` content. The client adds download buttons to the existing `EndGameOptions` component, triggering a client-side file download using the Blob API (following the existing CSV export pattern in `DeckPage`).

**Tech Stack:** TypeScript (server), React/Next.js/MUI (client), Socket.IO (transport), Jasmine (tests)

**Spec:** `docs/superpowers/specs/2026-04-04-swu-pgn-design.md`

// server/game/core/chat/SwuPgn.ts
import type { IPgnHeader, IPgnPlayerDecklist, IPgnCardIndexEntry, IPgnReplayRecord } from './PgnTypes';

export class SwuPgn {
    /**
     * Format the PGN header block as [Tag "Value"] lines.
     */
    public static formatHeader(header: IPgnHeader): string {
        const lines: string[] = [];

        const requiredTags: [string, keyof IPgnHeader][] = [
            ['Game', 'game'],
            ['Date', 'date'],
            ['Player1', 'player1'],
            ['Player2', 'player2'],
            ['P1Leader', 'p1Leader'],
            ['P1Base', 'p1Base'],
            ['P2Leader', 'p2Leader'],
            ['P2Base', 'p2Base'],
            ['Result', 'result'],
            ['Reason', 'reason'],
        ];

        const optionalTags: [string, keyof IPgnHeader][] = [
            ['Format', 'format'],
            ['Rounds', 'rounds'],
        ];

        for (const [tagName, key] of requiredTags) {
            lines.push(`[${tagName} "${header[key]}"]`);
        }

        for (const [tagName, key] of optionalTags) {
            if (header[key] != null) {
                lines.push(`[${tagName} "${header[key]}"]`);
            }
        }

        return lines.join('\n');
    }

    /**
     * Format the card index section with full decklists for both players.
     */
    public static formatCardIndex(p1Decklist: IPgnPlayerDecklist, p2Decklist: IPgnPlayerDecklist): string {
        const lines: string[] = [];
        lines.push('\u2550\u2550\u2550 CARD INDEX \u2550\u2550\u2550');
        lines.push('');
        lines.push(...SwuPgn.formatPlayerDecklist('P1', p1Decklist));
        lines.push('');
        lines.push(...SwuPgn.formatPlayerDecklist('P2', p2Decklist));

        return lines.join('\n');
    }

    private static formatPlayerDecklist(playerLabel: string, decklist: IPgnPlayerDecklist): string[] {
        const lines: string[] = [];
        lines.push(`\u2500\u2500 ${playerLabel} Decklist \u2500\u2500`);
        lines.push(`Leader: ${decklist.leader.name} = ${decklist.leader.setId}`);
        lines.push(`Base: ${decklist.base.name} = ${decklist.base.setId}`);
        lines.push('Deck:');

        const sortedDeck = [...decklist.deck].sort((a, b) => a.name.localeCompare(b.name));
        for (const entry of sortedDeck) {
            lines.push(`  ${entry.count}x ${entry.name} = ${entry.setId}`);
        }

        if (decklist.sideboard && decklist.sideboard.length > 0) {
            lines.push('Sideboard:');
            const sortedSideboard = [...decklist.sideboard].sort((a, b) => a.name.localeCompare(b.name));
            for (const entry of sortedSideboard) {
                lines.push(`  ${entry.count}x ${entry.name} = ${entry.setId}`);
            }
        }

        return lines;
    }

    /**
     * Format the replay data section as JSON-lines.
     */
    public static formatReplayData(records: IPgnReplayRecord[]): string {
        const lines: string[] = [];
        lines.push('=== REPLAY DATA ===');
        for (const record of records) {
            lines.push(JSON.stringify(record));
        }
        return lines.join('\n');
    }

    /**
     * Combine all sections into a complete .swupgn file.
     */
    public static formatFile(
        header: IPgnHeader,
        humanNotation: string,
        p1Decklist: IPgnPlayerDecklist,
        p2Decklist: IPgnPlayerDecklist,
        replayData: IPgnReplayRecord[]
    ): string {
        const sections: string[] = [];
        sections.push(SwuPgn.formatHeader(header));
        sections.push('');
        sections.push(humanNotation);
        sections.push('');
        sections.push(SwuPgn.formatCardIndex(p1Decklist, p2Decklist));
        sections.push('');
        sections.push(SwuPgn.formatReplayData(replayData));
        return sections.join('\n');
    }

    /**
     * Format a card's set ID as SET#NUM (zero-padded to 3 digits).
     */
    public static formatSetId(set: string, number: number): string {
        return `${set.toUpperCase()}#${String(number).padStart(3, '0')}`;
    }

    /**
     * Format a card's display name as "Title, Subtitle" or just "Title".
     */
    public static formatCardName(title: string, subtitle?: string): string {
        if (subtitle) {
            return `${title}, ${subtitle}`;
        }
        return title;
    }
}
```

    /**
     * Flatten a serialized message (which can be a string, array of mixed types, or alert object)
     * into a plain text string.
     */
    public static flattenMessage(message: any): string {
        if (typeof message === 'string') {
            return message;
        }

        if (typeof message === 'number') {
            return String(message);
        }

        if (message && typeof message === 'object' && 'alert' in message) {
            return SwuPgn.flattenMessage(message.alert.message);
        }

        if (Array.isArray(message)) {
            return message.map((part) => {
                if (typeof part === 'string' || typeof part === 'number') {
                    return String(part);
                }
                if (part && typeof part === 'object') {
                    if ('title' in part) {
                        if (part.subtitle) {
                            return `${part.title}, ${part.subtitle}`;
                        }
                        return part.title;
                    }
                    if ('name' in part) {
                        return part.name;
                    }
                }
                return '';
            }).join('');
        }

        if (message && typeof message === 'object' && 'name' in message) {
            return message.name;
        }

        return '';
    }
```

    /**
     * Replace player names with P1/P2 in a text line.
     * Handles both plain names and possessive forms (e.g., "Player's").
     */
    public static anonymizePlayers(text: string, player1Name: string, player2Name: string): string {
        // Replace possessive forms first (longer match) to avoid partial replacement
        let result = text;
        result = result.replaceAll(`${player1Name}'s`, "P1's");
        result = result.replaceAll(`${player2Name}'s`, "P2's");
        result = result.replaceAll(player1Name, 'P1');
        result = result.replaceAll(player2Name, 'P2');
        return result;
    }
```

    /**
     * Generate the human-readable notation body from game chat messages.
     * Flattens each message to text and anonymizes player names.
     * Excludes player chat messages (only game log entries).
     */
    public static generateHumanNotation(
        messages: any[],
        player1Name: string,
        player2Name: string
    ): string {
        const lines: string[] = [];

        for (const entry of messages) {
            const message = entry.message;

            // Skip player chat messages
            if (SwuPgn.isPlayerChatMessage(message)) {
                continue;
            }

            const flatText = SwuPgn.flattenMessage(message);
            if (!flatText || flatText.trim() === '') {
                continue;
            }

            const anonymized = SwuPgn.anonymizePlayers(flatText, player1Name, player2Name);
            lines.push(anonymized);
        }

        return lines.join('\n');
    }

    /**
     * Check if a message is a player chat message (not a game log entry).
     */
    private static isPlayerChatMessage(message: any): boolean {
        if (Array.isArray(message) && message.length > 0) {
            const firstElement = message[0];
            if (typeof firstElement === 'object' && firstElement && 'type' in firstElement && firstElement.type === 'playerChat') {
                return true;
            }
        }
        return false;
    }
```

    /**
     * Returns the full game log as plain text with player names anonymized.
     * Excludes player chat messages.
     */
    public getRawGameLog(): string {
        const players = this.getPlayers();
        const player1Name = players[0].name;
        const player2Name = players[1].name;

        return SwuPgn.generateHumanNotation(this.gameChat.messages, player1Name, player2Name);
    }

    /**
     * Generate a complete .swupgn file for this game.
     * Should only be called after the game has ended.
     */
    public generateSwuPgn(): string {
        const players = this.getPlayers();
        const player1 = players[0];
        const player2 = players[1];

        const header = this.buildPgnHeader(player1, player2);
        const humanNotation = this.getRawGameLog();
        const p1Decklist = this.buildPlayerDecklist(player1);
        const p2Decklist = this.buildPlayerDecklist(player2);

        // For v1.0, replay data is empty — will be populated in a future iteration
        // when we add structured event recording
        const replayData: any[] = [];

        return SwuPgn.formatFile(header, humanNotation, p1Decklist, p2Decklist, replayData);
    }

    private buildPgnHeader(player1: Player, player2: Player): IPgnHeader {
        const winner = this.winnerNames;
        let result: string;
        if (winner.length === 0) {
            result = 'Incomplete';
        } else if (winner.length > 1) {
            result = 'Draw';
        } else if (winner[0] === player1.name) {
            result = 'P1 Win';
        } else {
            result = 'P2 Win';
        }

        let reason: string;
        switch (this.gameEndReason) {
            case GameEndReason.Concede:
                reason = 'Concession';
                break;
            case GameEndReason.PlayerLeft:
                reason = 'Disconnection';
                break;
            case GameEndReason.GameRules:
            default:
                reason = 'Base Destroyed';
                break;
        }

        return {
            game: 'SWU-PGN v1.0',
            date: new Date().toISOString().slice(0, 10).replaceAll('-', '.'),
            player1: 'Player 1',
            player2: 'Player 2',
            p1Leader: SwuPgn.formatCardName(player1.leader.title, player1.leader.subtitle),
            p1Base: SwuPgn.formatCardName(player1.base.title, player1.base.subtitle),
            p2Leader: SwuPgn.formatCardName(player2.leader.title, player2.leader.subtitle),
            p2Base: SwuPgn.formatCardName(player2.base.title, player2.base.subtitle),
            result,
            reason,
            rounds: String(this.roundNumber),
        };
    }

    private buildPlayerDecklist(player: Player): IPgnPlayerDecklist {
        const leader: IPgnCardIndexEntry = {
            name: SwuPgn.formatCardName(player.leader.title, player.leader.subtitle),
            setId: SwuPgn.formatSetId(player.leader.setId.set, player.leader.setId.number),
            count: 1,
        };

        const base: IPgnCardIndexEntry = {
            name: SwuPgn.formatCardName(player.base.title, player.base.subtitle),
            setId: SwuPgn.formatSetId(player.base.setId.set, player.base.setId.number),
            count: 1,
        };

        // Build deck entries from all cards in the deck zone (excluding leader, base, and tokens)
        const cardCounts = new Map<string, IPgnCardIndexEntry>();
        for (const card of player.allCards) {
            // Skip leader, base, and token cards
            if (card === player.leader || card === player.base || card.isToken()) {
                continue;
            }
            const setId = SwuPgn.formatSetId(card.setId.set, card.setId.number);
            const existing = cardCounts.get(setId);
            if (existing) {
                existing.count++;
            } else {
                cardCounts.set(setId, {
                    name: SwuPgn.formatCardName(card.title, card.subtitle),
                    setId,
                    count: 1,
                });
            }
        }

        return {
            leader,
            base,
            deck: Array.from(cardCounts.values()),
        };
    }

    public getGameLog(socket: any, callback: (data: { rawLog: string; swuPgn: string } | { error: string }) => void): void {

// forceteki-client/src/app/_components/_sharedcomponents/Preferences/_subComponents/DownloadGameLog.tsx
import React, { useState } from 'react';
import Box from '@mui/material/Box';
import Typography from '@mui/material/Typography';
import PreferenceButton from './PreferenceButton';
import { useGame } from '@/app/_contexts/Game.context';

function DownloadGameLog() {
    const { requestGameLog } = useGame();
    const [loading, setLoading] = useState(false);
    const [downloaded, setDownloaded] = useState<'none' | 'log' | 'pgn' | 'both'>('none');

    const triggerDownload = (content: string, filename: string, mimeType: string) => {
        const blob = new Blob([content], { type: `${mimeType};charset=utf-8;` });
        const url = URL.createObjectURL(blob);
        const a = document.createElement('a');
        a.href = url;
        a.download = filename;
        a.click();
        URL.revokeObjectURL(url);
    };

    const handleDownloadLog = async () => {
        setLoading(true);
        try {
            const data = await requestGameLog();
            if (data) {
                triggerDownload(data.rawLog, 'game-log.txt', 'text/plain');
                setDownloaded((prev) => prev === 'pgn' ? 'both' : 'log');
            }
        } finally {
            setLoading(false);
        }
    };

    const handleDownloadPgn = async () => {
        setLoading(true);
        try {
            const data = await requestGameLog();
            if (data) {
                triggerDownload(data.swuPgn, 'game.swupgn', 'text/plain');
                setDownloaded((prev) => prev === 'log' ? 'both' : 'pgn');
            }
        } finally {
            setLoading(false);
        }
    };

    const styles = {
        contentContainer: {
            display: 'flex',
            flexDirection: 'row' as const,
            alignItems: 'center',
            mb: '20px',
        },
        typeographyStyle: {
            ml: '2rem',
            color: '#878787',
            lineHeight: '15.6px',
            size: '13px',
            weight: '500',
        },
    };

    return (
        <>
            <Box sx={styles.contentContainer}>
                <PreferenceButton
                    variant={'standard'}
                    text={loading ? 'Loading...' : 'Download Game Log'}
                    buttonFnc={handleDownloadLog}
                    disabled={loading}
                />
                <Typography sx={styles.typeographyStyle}>
                    Download the raw game log as a text file.
                </Typography>
            </Box>
            <Box sx={styles.contentContainer}>
                <PreferenceButton
                    variant={'standard'}
                    text={loading ? 'Loading...' : 'Download SWU-PGN'}
                    buttonFnc={handleDownloadPgn}
                    disabled={loading}
                />
                <Typography sx={styles.typeographyStyle}>
                    Download game notation for replay and analysis.
                </Typography>
            </Box>
        </>
    );
}

export default DownloadGameLog;

## Implementation Notes

### What's included in v1.0
- Raw game log download (anonymized, plain text)
- SWU-PGN file with header, human notation (from existing game messages), card index (from decklists), and replay data section (initially sparse — populated from what's available in game messages)
- Download buttons in the end game modal

### What's deferred to future iterations
- **Structured replay data**: The current game engine doesn't emit structured events in a format ready for the replay JSON-lines. The `=== REPLAY DATA ===` section will initially contain basic records. Full structured event recording would require instrumenting the game event system (`GameEvent.ts`) to emit `IPgnReplayRecord` entries alongside the existing message log — a larger change that should be its own spec/plan.
- **Round/phase structure markers**: The existing game messages don't include explicit round/phase boundary markers in a consistent parsable format. The human notation in v1.0 will be the anonymized flat message stream. Adding `═══ ROUND N ═══` and `─── Phase ───` markers requires either (a) adding explicit phase-transition messages in the game engine, or (b) detecting phase transitions from message content patterns — both are future enhancements.
- **Action numbering and sub-event hierarchy**: Similarly deferred — requires tracking which messages are triggered by which actions, which the current flat message stream doesn't capture.
