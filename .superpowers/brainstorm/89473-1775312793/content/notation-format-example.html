Key Format Decisions Shown

  • Header: Chess PGN-style [Tag "Value"] metadata block
  • Round markers: ═══ ROUND N ═══ with phase sub-headers
  • Action numbering: Resets per action phase. Sub-events use 1a, 1b, 1c
  • Player references: Always P1 / P2
  • Card names: Full title as written on card (human-readable in the notation, unique ID in the index)
  • Hidden info revealed: Resourced cards named, hands shown when looked at
  • Full cascade: Every trigger, damage tick, state change gets its own sub-line
  • Card Index: Full decklists grouped by player with count x Name = SET#NUM
