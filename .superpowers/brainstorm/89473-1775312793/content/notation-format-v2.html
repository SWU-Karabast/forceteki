Naming Convention

Card Type Has Subtitle? Notation Example
Named Unit ✓ Yes Admiral Ozzel, Overconfident
Generic Unit ✗ No Cell Block Guard
Leader ✓ Yes Darth Vader, Commanding the First Legion
Event ✗ No Vanquish
Upgrade Sometimes Experience Token or Darksaber, Ancient Weapon
Base ✗ No Echo Base
Token Unit ✗ No X-Wing token

Rule: Always use "Title, Subtitle" when a subtitle exists. Use just "Title" when there is no subtitle. This applies everywhere — notation body and card index.
